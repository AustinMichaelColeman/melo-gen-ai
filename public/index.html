Smart Playlist Generator

A plugin for ChatGPT and for YouTube Music™

Enables playlist generation for ChatGPT and for YouTube Music™.

Project Features

  • Generates a custom playlist on YouTube based on interacting with ChatGPT.
  • Asks user's consent before generating the public playlist.
  • Deployed with Vercel serverless functions
