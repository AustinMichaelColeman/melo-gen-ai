MeloGenAI

A playlist generator plugin for ChatGPT and for YouTube Music™

Enables playlist generation for ChatGPT and for YouTube Music™.

Project Features

  • Generates a playlist on the user's YouTube Music™ account through the ChatGPT plugin.
  • Searches YouTube for each song title provided, adding the first returned result to the playlist.
  • Prioritizes user privacy by seeking consent before starting the song search and playlist creation process.
  • Allows the user to specify public, private, or unlisted for their playlist.
